# Guia de Imagens - Burguerlito

Coloque as imagens dos produtos na pasta `images/`, na raiz do projeto.

```text
burguerlito/
  html/
  css/
  js/
  images/
    smash-simples.png
    smash-duplo.png
    x-salada-smash.png
  README.md
```

## Nomes esperados

### Lanches

```text
smash-simples.png
smash-duplo.png
x-salada-smash.png
```

### Combos

```text
combo-essencial-1.png
combo-duplo-brutal-2.png
combo-fresh-3.png
combo-casal-4.png
combo-casal-premium-5.png
combo-fome-de-verdade-6.png
combo-trio-7.png
```

### Bebidas

```text
agua-500ml.png
refrigerante-sprite.png
refrigerante-fanta-laranja.png
refrigerante-pepsi.png
refrigerante-fanta-uva.png
```

## Formato recomendado

- Extensão: JPG ou PNG.
- Dimensão mínima: 400x400px.
- Dimensão recomendada: 600x600px ou maior.
- Tamanho ideal: até 500KB por imagem.
- Proporção: quadrada ou horizontal bem centralizada.

## Como atualizar

1. Exporte a imagem final com o nome exato do produto.
2. Coloque o arquivo dentro de `images/`.
3. Abra `html/index.html` no navegador.
4. Recarregue a página.

## Se uma imagem não carregar

- Confirme se o nome está igual ao indicado neste arquivo.
- Confirme se a imagem está dentro de `images/`.
- Confirme se o caminho em `js/menu.js` aponta para o arquivo correto.
- Evite espaços e caracteres especiais no nome do arquivo.

Enquanto a imagem final não existir, o card mostra um placeholder com o texto `Imagem em breve`.
